🔗 Link e Navigazione

Impariamo a collegare le pagine e creare menu di navigazione efficaci

🤔 Cos'è un Link?

Un link (o collegamento) è un elemento che permette di navigare da una pagina all'altra, o verso risorse esterne. È quello che rende il web "interconnesso"!

💻 Codice HTML
<a href="https://www.google.com">Vai su Google</a> <a href="pagina2.html">Vai alla Pagina 2</a> <a href="mailto:info@scuola.it">Invia Email</a> <a href="tel:+393331234567">Chiama</a>
👀 Risultato Visibile

Vai su Google

Vai alla Pagina 2

Invia Email

Chiama

💡 Anatomia di un Link

<a href="destinazione">Testo visibile</a>
• href = dove andare quando si clicca
• Testo visibile = quello che l'utente vede e può cliccare

🎯 Tipi di Link

Esistono diversi tipi di link, ognuno con uno scopo specifico:

Link Esterni

Verso altri siti web

<a href="https://www.google.com">Google</a> <a href="https://www.youtube.com">YouTube</a>
🌐 Vai su Google
📺 Vai su YouTube
💡 Si aprono in una nuova scheda con target="_blank"
Link Interni

Verso altre pagine del sito

<a href="about.html">Chi Siamo</a> <a href="contatti.html">Contatti</a> <a href="../index.html">Home</a>
🏠 Torna alla Home
🎨 Lezione CSS
💡 Usano percorsi relativi o assoluti
Link Email

Per aprire il client email

<a href="mailto:prof@scuola.it">Email al Prof</a> <a href="mailto:info@scuola.it?subject=Domanda"> Email con Oggetto </a>
📧 Email al Professore
📨 Email con Oggetto
💡 Apre automaticamente il programma email
Link Telefono

Per chiamare su mobile

<a href="tel:+393331234567">Chiama</a> <a href="tel:0123456789">Numero Fisso</a>
📱 Chiama Mobile
☎️ Chiama Fisso
💡 Funziona solo su dispositivi con telefono
Link Anchor

Verso sezioni della stessa pagina

<a href="#sezione1">Vai alla Sezione 1</a> <a href="#top">Torna su</a> <h2 id="sezione1">Sezione 1</h2>
⬇️ Vai ai Menu
⬆️ Torna su
💡 Usa # seguito dall'ID dell'elemento
Link File

Per scaricare documenti

<a href="documento.pdf">Scarica PDF</a> <a href="foto.jpg" download>Scarica Foto</a>
📄 Scarica PDF
🖼️ Scarica Immagine
💡 L'attributo download forza il download

⚙️ Attributi Importanti dei Link

I link hanno diversi attributi che ne modificano il comportamento:

📋 Attributi Principali
<!-- Apre in nuova scheda --> <a href="https://google.com" target="_blank"> Google (nuova scheda) </a> <!-- Sicurezza per link esterni --> <a href="https://sito-esterno.com" target="_blank" rel="noopener noreferrer"> Link Sicuro </a> <!-- Titolo al passaggio del mouse --> <a href="pagina.html" title="Vai alla pagina informazioni"> Info </a> <!-- Forza il download --> <a href="file.pdf" download="nome-file.pdf"> Scarica </a>
🎯 Spiegazione

target="_blank"
Apre il link in una nuova scheda/finestra

rel="noopener noreferrer"
Migliora la sicurezza per link esterni

title="..."
Mostra un tooltip al passaggio del mouse

download="nome"
Forza il download invece di aprire il file

🔗 Prova questo link sicuro

⚠️ Sicurezza

Usate sempre rel="noopener noreferrer" con target="_blank" per link esterni. Previene attacchi di sicurezza!

🧭 Menu di Navigazione

I menu sono collezioni di link organizzati per aiutare gli utenti a navigare nel sito:

📱 Menu Orizzontale

<nav> <ul class="menu-horizontal"> <li><a href="index.html">Home</a></li> <li><a href="about.html">Chi Siamo</a></li> <li><a href="servizi.html">Servizi</a></li> <li><a href="contatti.html">Contatti</a></li> </ul> </nav>
  • Home
  • Chi Siamo
  • Servizi
  • Contatti

📋 Menu Verticale

<nav> <ul class="menu-vertical"> <li><a href="lezione1.html">Lezione 1: HTML Base</a></li> <li><a href="lezione2.html">Lezione 2: CSS Base</a></li> <li><a href="lezione3.html">Lezione 3: Link</a></li> <li><a href="lezione4.html">Lezione 4: Layout</a></li> </ul> </nav>
  • Lezione 1: HTML Base
  • Lezione 2: CSS Base
  • Lezione 3: Link
  • Lezione 4: Layout

💡 Best Practice per Menu

  • Usate sempre il tag <nav> per i menu principali
  • Organizzate i link in liste <ul> e <li>
  • Rendete chiaro quale pagina è attiva
  • Testate sempre su mobile!

🎨 Stili CSS per Link

I link hanno 4 stati diversi che potete stilizzare:

💻 CSS per Stati Link
/* Link normale */ a { color: #3498db; text-decoration: none; transition: all 0.3s ease; } /* Link visitato */ a:visited { color: #9b59b6; } /* Link al passaggio del mouse */ a:hover { color: #e74c3c; text-decoration: underline; transform: translateY(-1px); } /* Link mentre viene cliccato */ a:active { color: #27ae60; transform: translateY(0); }

🚀 Esercizio Pratico

Ora tocca a voi! Create una pagina con un menu di navigazione completo.

📝 Cosa fare:

  1. Create un nuovo file chiamato menu-test.html
  2. Aggiungete un menu orizzontale con almeno 5 link
  3. Includete diversi tipi di link (interno, esterno, email)
  4. Stilizzate i link con CSS (colori, hover effects)
  5. Testate tutti i link per assicurarvi che funzionino

🎯 Sfida Bonus:

  • Aggiungete un menu verticale nella sidebar
  • Create un "breadcrumb" (percorso di navigazione)
  • Aggiungete animazioni ai link con CSS
  • Rendete il menu responsive per mobile
