🎨 Stili CSS

Impariamo a dare colore, forma e bellezza alle nostre pagine web

🤔 Cos'è il CSS?

CSS significa Cascading Style Sheets (Fogli di Stile a Cascata). È il linguaggio che dice al browser come far apparire gli elementi HTML.

📄 HTML Senza CSS
<h1>Titolo</h1>
<p>Paragrafo di testo</p>

Titolo

Paragrafo di testo

→ Brutto, vero? Tutto nero, font predefinito, nessun stile.

✨ HTML Con CSS
h1 { color: blue; font-size: 2rem; }
p { color: gray; line-height: 1.5; }

Titolo

Paragrafo di testo

→ Molto meglio! Colori, dimensioni, spaziature personalizzate.

💡 Analogia Utile

Pensate all'HTML come allo scheletro di una casa (struttura) e al CSS come alla decorazione (colori, mobili, stile).

🔍 Anatomia di una Regola CSS

Ogni regola CSS ha sempre la stessa struttura. Impariamo a leggerla:

h1 { /* <-- SELETTORE: dice "a quale elemento applicare lo stile" */
  color: blue; /* <-- PROPRIETÀ: VALORE */
  font-size: 2rem; /* <-- Ogni proprietà finisce con ; */
  margin-bottom: 1rem;
} /* <-- Chiudere sempre con } */

🎯 Le Parti:

  • SELETTORE (h1) = "Applica questo stile a tutti gli h1"
  • PROPRIETÀ (color, font-size) = "Cosa vuoi cambiare"
  • VALORE (blue, 2rem) = "Come vuoi che diventi"
  • ; = "Ho finito questa proprietà"
  • { } = "Tutto quello dentro si applica al selettore"

🎯 I Selettori CSS Fondamentali

I selettori dicono al CSS su quale elemento applicare lo stile. Ci sono tre tipi principali:

🏷️ Selettore per TAG

h1 { color: red; }

Significa: "Applica a tutti gli elementi h1"

HTML coinvolto: <h1>Titolo</h1>

Quando usarlo: Per stilizzare tutti gli elementi di quel tipo

🆔 Selettore per ID

#menu { background: blue; }

Significa: "Applica all'elemento con id='menu'"

HTML coinvolto: <div id="menu">...</div>

Quando usarlo: Per un elemento unico e specifico

🔖 Selettore per CLASSE

.bottone { padding: 10px; }

Significa: "Applica a tutti gli elementi con class='bottone'"

HTML coinvolto: <a class="bottone">Click</a>

Quando usarlo: Per più elementi simili

🎲 Come Ricordare i Simboli

# = "hashtag" = unico → ID
. = "punto" = gruppo → CLASS
niente = tutti quelli del tag

📝 Dove Scrivere il CSS?

Ci sono tre modi per aggiungere CSS alle vostre pagine. Vediamo quando usare ciascuno:

🎯 CSS Inline (nel tag)

<h1 style="color: red; font-size: 2rem;">Titolo</h1>

Pro: Veloce per test rapidi

Contro: Difficile da gestire, non riutilizzabile

Quando usarlo: Solo per esperimenti o correzioni veloci

📄 CSS Interno (nel file HTML)

<style>
h1 { color: red; }
p { font-size: 18px; }
</style>

Pro: Tutto in un unico file

Contro: Diventa confuso se il sito cresce

Quando usarlo: Per piccole pagine o esercizi

📂 CSS Esterno (file separato)

<link rel="stylesheet" href="stili.css">

Pro: Ordinato, riutilizzabile, professionale

Contro: Serve gestire più file

Quando usarlo: Sempre in progetti veri!

💡 Consiglio

Nei vostri progetti scolastici imparate subito a usare un file CSS esterno: è l’approccio che userete sempre nel mondo reale.
