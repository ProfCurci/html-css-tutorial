🚀 Setup dell'Ambiente di Sviluppo

Installiamo Visual Studio Code e creiamo il nostro primo file HTML

1

Download di Visual Studio Code

Visual Studio Code è un editor gratuito e potente, perfetto per lo sviluppo web. È quello che usano anche i programmatori professionali!

Come scaricare:

  1. Vai sul sito ufficiale: code.visualstudio.com
  2. Clicca sul pulsante blu "Download for Windows" (o Mac/Linux)
  3. Salva il file sul tuo computer
📸 Screenshot: Pagina di download di VS Code con il pulsante evidenziato
📥 Scarica Visual Studio Code

⚠️ Attenzione

Assicuratevi di scaricare dal sito ufficiale Microsoft. Evitate siti di terze parti!

2

Installazione di Visual Studio Code

Procedura di installazione:

  1. Esegui il file scaricato (VSCodeUserSetup-x64-x.xx.x.exe)
  2. Accetta i termini di licenza
  3. IMPORTANTE: Seleziona queste opzioni:
  • Add "Open with Code" action to Windows Explorer file context menu
  • Add "Open with Code" action to Windows Explorer directory context menu
  • Register Code as an editor for supported file types
  • Add to PATH
📸 Screenshot: Finestra di installazione con le opzioni da selezionare evidenziate
Queste opzioni vi permetteranno di aprire file e cartelle direttamente con VS Code!

✅ Installazione completata!

Se tutto è andato bene, dovreste vedere l'icona di VS Code sul desktop o nel menu Start.

3

Primo Avvio e Configurazione Base

Apri Visual Studio Code:

  1. Clicca sull'icona di VS Code
  2. Al primo avvio, potresti vedere una schermata di benvenuto
  3. Chiudi eventuali tab di benvenuto (X sulla linguetta)

Estensioni utili (opzionale ma consigliato):

  1. Clicca sull'icona delle estensioni nella barra laterale (quadrati)
  2. Cerca e installa queste estensioni:
  • Live Server - per vedere le pagine in tempo reale
  • HTML CSS Support - per aiuto con il codice
  • Prettier - per formattare automaticamente il codice
📸 Screenshot: VS Code aperto con la sezione estensioni visibile
4

Crea la Tua Prima Pagina HTML

Creazione del file:

  1. In VS Code, clicca su File → New File
  2. Salva subito il file: Ctrl+S (o Cmd+S su Mac)
  3. Chiamalo index.html
  4. Scegli dove salvarlo (meglio creare una cartella "HTML-Tutorial")

⚠️ Nome del file

È importante che il file finisca con .html - questo dice al computer che è una pagina web!

📄 index.html
<!DOCTYPE html> <html lang="it"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>La Mia Prima Pagina!</title> <style> body { font-family: Arial, sans-serif; max-width: 800px; margin: 0 auto; padding: 2rem; background-color: #f0f8ff; } h1 { color: #2c3e50; text-align: center; border-bottom: 3px solid #3498db; padding-bottom: 1rem; } .success-message { background: #d4edda; border: 1px solid #c3e6cb; padding: 1rem; border-radius: 5px; margin: 2rem 0; } .code-info { background: #f8f9fa; border-left: 4px solid #007bff; padding: 1rem; margin: 1rem 0; } </style> </head> <body> <h1>🎉 Congratulazioni!</h1> <div class="success-message"> <h2>✅ Hai creato la tua prima pagina web!</h2> <p>Se stai vedendo questa pagina nel browser, significa che tutto funziona correttamente.</p> </div> <h2>Cosa hai imparato:</h2> <ul> <li>Come installare Visual Studio Code</li> <li>Come creare un file HTML</li> <li>Come aprire una pagina web nel browser</li> </ul> <div class="code-info"> <h3>💡 Prova a modificare qualcosa!</h3> <p>Cambia il testo di questo paragrafo, salva il file (Ctrl+S) e ricarica la pagina nel browser (F5). Vedrai le tue modifiche!</p> </div> <h2>Prossimo passo:</h2> <p>Ora sei pronto per imparare la struttura HTML e come modificare gli stili con CSS!</p> <footer style="margin-top: 3rem; text-align: center; color: #666;"> <p>Prima pagina creata con Visual Studio Code - Tutorial HTML & CSS</p> </footer> </body> </html>

Cosa fare:

  1. Copia tutto il codice qui sopra
  2. Incolla nel tuo file index.html in VS Code
  3. Salva il file (Ctrl+S)
5

Aprire la Pagina nel Browser

Metodo 1 - Trascinamento:

  1. Apri una finestra del browser (Chrome, Firefox, Edge...)
  2. Apri la cartella dove hai salvato index.html
  3. Trascina il file index.html nella finestra del browser

Metodo 2 - Da VS Code (se hai installato Live Server):

  1. Tasto destro sul file index.html in VS Code
  2. Scegli "Open with Live Server"
  3. Si aprirà automaticamente nel browser

🎯 Obiettivo raggiunto!

Dovreste vedere una bella pagina colorata con congratulazioni. Se vedete il codice scritto invece della pagina, ricontrollate che il file finisca con .html

Test finale:

  1. Nella pagina web, cambiate il titolo da "La Mia Prima Pagina!" a qualcosa di vostro
  2. Salvate il file in VS Code (Ctrl+S)
  3. Ricaricate la pagina nel browser (F5)
  4. Il titolo dovrebbe cambiare!
